OpenMobile AppMall Archive

Welcome to the community-maintained archive of the OpenMobile AppMall for ACL (Application Compatibility Layer).

What is this?
This server provides the backend API for the AppMall app that comes bundled with ACL on HP TouchPad. The original OpenMobile servers shut down years ago, so this community archive keeps the AppMall functional.

For HP TouchPad Users

If you have ACL installed on your HP TouchPad, the AppMall app should now work automatically. Open the AppMall to browse and download apps that have been tested to work with ACL.

Available Apps

This archive includes a curated selection of Android apps compatible with Android 2.3 (Gingerbread) that run well on ACL:

  • File Managers - ES File Explorer, and more
  • Browsers - Opera Mini, Firefox
  • Media Players - VLC, MX Player
  • Productivity - Kingsoft Office, Adobe Reader
  • Games - Angry Birds, Cut the Rope, and classics

Contributing Apps

Want to add an app?
We're looking for APKs of apps that work well on Android 2.3/ACL. If you have suggestions or APK files to contribute, please reach out to the community.

Technical Information

This server implements the OpenMobile AppMall API:

  • API Endpoint: /appmallpipe.php
  • Response Format: XML
  • Authentication: POS ID based (handled automatically by AppMall app)

About ACL

ACL (Application Compatibility Layer) was developed by OpenMobile to run Android 2.3 (Gingerbread) apps on HP TouchPad running webOS 3.0.x. While OpenMobile is no longer in business, the ACL software continues to work thanks to community-maintained patches.

This is a community preservation project.
